Unix Commands

I was practicing the Unix commands and i have found an interesting command.

Hiding :

To create a file/directory as a hidden file/directory , just add dot (.) operator in front of the file name. Thats gives you the hidden file.

Example:

$touch .filename.txt # This is example to create the hidden file
$mkdir .directory # This is example to create the hidden directory
To view the hidden files in unix,
$ls -al #This displays all the hidden files/directories

UnHiding :

To make hidden file unhide, we need to simply rename the file using the mv command.

Example:

$mv .hidden-file.txt new-file.txt #This is the example for a file
$mv .hidden-directory unhidden-directory

DIRECTORY STACK STRUCTURE FOR PUSHD & POPD :

Please pay attention on this topic, it may be bit confusing at the beginning but it is all about stack.

Example :

Create a directory as below ,
$mkdir -p A/B/C/D/E ----------->1
$cd A ----------->2
$pushd B ----------->3
$cd C ----------->4
$pushd D ----------->5
$cd E ----------->6
$pwd
You will be in the last directory E.
$popd ------------->7
$popd ------------->8
$popd [ This will return a warning message like ] bash: popd: directory stack empty ] ------------->9
